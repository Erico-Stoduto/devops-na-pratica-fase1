# DevOps na Prática - Fase 1

Este projeto foi desenvolvido para a disciplina DevOps na Prática e tem como objetivo aplicar, de forma prática, conceitos de automação, integração contínua e infraestrutura como código.

## Descrição do Projeto
Foi criado um site estático simples, armazenado no repositório e versionado utilizando Git.  
A hospedagem do site é realizada na AWS S3, e toda a infraestrutura é provisionada automaticamente com **Terraform**.

Além disso, foi configurado um **pipeline de Integração Contínua (CI)** utilizando **GitHub Actions**, que valida automaticamente o projeto a cada atualização enviada para a branch `main`.

## Tecnologias Utilizadas
- Git e GitHub
- GitHub Actions (CI)
- AWS S3 (Hospedagem)
- Terraform (Infraestrutura como Código)
- HTML/CSS
